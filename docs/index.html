TeX to Unicode

Input Unicode mathematical symbols by LaTeX

A Chrome extension

Convert by pressing Alt + w (or ^ + w on Mac).

Add to Chrome

Supported symbols

Features

  • Convert Tex to Unicode by pressing ALT + W (OPT + W on Mac)
  • Convert underscripts and superscripts (can be enabled by an option)
  • Works on Stack Exchange, Reddit, Wikipedia, and Gmail
  • Convert one symbol, or convert all symbols under selection
  • Free software
