The traditional way to input Unicode is intense copy paste. With TeX to Unicode you can input Unicode by it's LaTex name, and then convert the LaTex to Unicode by pressing Alt + w. The use cases includes Stack Exchange, Reddit, Wikipedia and Gmail. In sites that already accepted LaTex input through MathJax, you can still mix LaTex with Unicode to improve readability.

Place keyboard cursor at the LaTex symbol, and then press Alt + W.

Multiple convert by selecting multiple symbols.
